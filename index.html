O que é o HTML

O HTML é uma linguagem de marcação de texto, essencial para a criação de aplicações web. Através dele conseguimos estruturar uma página web, adequando os componentes visuais de acordo com a prosposta desejada. Mesmo não sendo uma linguagem de programação, possui regras de sintaxe e palavras reservadas de modo semelhante.

História do HTML

O HTML foi criado em 1991, por Tim Berners-Lee, no CERN (European Council for Nuclear Research) na suíça. Inicialmente o HTML foi projetado para interligar instituições de pesquisa próximas, e compartilhar documentos com facilidade. Em 1992, foi liberada a biblioteca de desenvolvimento WWW ( World Wide Web), uma rede de alcance mundial, que junto com o HTML proporcionou o uso em escala mundial da WEB.

Estruturas Usadas em HTML

A estrutura de um documento HTML é composta por três tags: html, head e body. As tags são escritas dentro dos sinais < e >. A seguir veremos detalhes sobre essas tags:

Tags html e head

A tag html é a criação do documento, onde ficarão as outras duas tags. Na tag head, colocaremos a informação que aparece na aba do navegador, geralmente o nome da página. Teremos também alguns elementos que não serão mostrados aos usuários, mas que são importantes para o navegador e outros serviços da web.

Tag body

Nesta tag teremos todos os elementos que aparecerão no website, tudo o que for visível ao usuário e o conteúdo da página.

Outras Tags

Existem muitas outras tags dentro do HTML, para inserir título, links, imagens, video e outros ítens.
